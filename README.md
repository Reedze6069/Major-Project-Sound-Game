Horror Sound Game

Horror Sound Game is a small 2D horror runner made in Unity.
The player moves forward automatically, and the main actions are controlled using sound.

## How to Play

Use your voice to control the sound bar:

* **Green** = Crouch
* **Orange** = Light / Reveal
* **Red** = Jump

Try to control your voice level and stay in the right colour depending on what you need to do.

The goal is to survive the level by using sound to avoid obstacles, reveal what is ahead, and react in time.

## Notes

For the best experience, use a proper microphone if possible. Earbuds can work, but they may pick up background noise or make the controls less reliable.
